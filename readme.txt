== Description ==

This plugin adds a hierarchical navigation to your `Pages` menu. This enables you to quickly find the page you want to edit.

Simply activate it to get it working!

Note: It loads the pages with AJAX so there might be a slight delay until the pages shows up.

== Installation ==

1. Upload `quickfind` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
